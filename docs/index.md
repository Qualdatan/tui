# qualdatan-tui

Terminal-GUI für [Qualdatan](https://github.com/GeneralPawz/Qualdatan). Typer-basierte CLI mit Rich-Output, die [qualdatan-core](https://github.com/Qualdatan/core) und [qualdatan-plugins](https://github.com/Qualdatan/plugins) orchestriert.

## Install

```bash
pip install qualdatan-tui
qualdatan --help
```
